The Kitten Programming Language

Kitten is a statically typed, stack-based functional programming language designed to be simple and fast. It is a concatenative language, combining aspects of imperative and pure functional programming. There is a short introduction available and a tutorial in progress.

Example

The source code is hosted on GitHub, and you can join #concatenative on Freenode to keep up with the latest developments.
